# Expense Tracker Full Stack Web Application

Track, manage, and visualize your expenses easily with this fullstack web app.

## Features

- User authentication for secure access
- Add, edit, delete, and view expenses
- Categorize expenses for better tracking
- Dashboard with summary and expense history
- Interactive charts to visualize spending trends
- Responsive UI

## Tech Stack

- **Frontend:** React.js, JavaScript, Chart.js, CSS
- **Backend:** Flask (Python)
- **Database:** MySQL

## Installation

### 1. Clone the repo

git clone https://github.com/madany044/Expense-Tracker.git
cd Expense-Tracker

### 2. Backend Setup

cd backend
pip install -r requirements.txt
python app.py

### 3. Frontend Setup

cd frontend
npm install
npm start
